A Single Page Application, using a single div with an id to allow the pages to re-render without reloading the page.
The entire project was written in Vanilla Javascript, using jQuery, HTML and SCSS complied into CSS and used Bootstrap to accomplish page structure.

Furthermore, I also used Rellax to create parallax effect on each page where the hero image is.
I managed the Crypto API in a dedicated object called CryptoManager which allows me to re-use whenever needed through the development.
Live Report page shows a chat that was build using Chart.jsThe chart sets all the prices to 0 and it updates every 2 seconds, It also dynamically updates the chart when the user toggles a coin.
